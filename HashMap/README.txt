**Όνομα**

HashMap

**Έκδοση**

Version: 0.9.0

**Τι είναι το HashMap;**

Το HashMap είναι μία εφαρμογή που χειρίζεται δομές δεδομένων για την αποθήκευση συνόλων στοιχείων

**Χειρισμός της εφαρμογής**

Για να χειριστείτε την εφαρμογή θα πρέπει να ανοίξετε την κονσόλα. (π.χ Windows PowerShell, Command Prompt)

**Μεταγλώττιση της εφαρμογής**

Για να μεταγλωττίσετε την εφαρμογή θα πρέπει να πληκτρολογήσετε στη κονσόλα:
"cd <HashMap_Path>"
και έπειτα 
"mvn package"
Κατά την ολοκλήρωση της μεταγώττισης θα δημιουργηθεί ένας κατάλογος "target" και μέσα σε αυτόν βρίσκεται 
το εκτελέσιμο αρχείο jar. 

<HashMap_Path>: Το Path όπου βρίσκονται μέσα τα αρχεία του HashMap (π.χ C:\Users\mpampis\Downloads\HashMap).

**Πώς εκτελώ την εφαρμογή;**

Για να εκτελέσετε την εφαρμογή πρέπει να ανοίξετε την κονσόλα και να πληκτρολογήσετε:
"java -cp <jar_file_path> data.structures.hashmap.HashMap"

Σημείωση: Απαιτούνται επιπλέον ορίσματα για να έχουμε output.

<jar_File_Path>: Το Path του jar file. (Δείτε **Μεταγλώττιση της εφαρμογής**)(είτε Relative είτε Full Path)

**Πώς τη χρησιμοποιώ;**

Κατά την εκτέλεση της εφαρμογής θα πρέπει να δώσετε ένα ή περισσότερα ορίσματα (π.χ java -cp mpampis.jar org.hua.hash.App [argument1] [argument2] ...)

Η εφαρμογή απαιτεί τουλάχιστον ένα όρισμα για να εμφανίσει output και αυτό θα πρέπει να είναι αρχείο κειμένου (txt file) για να λειτουργήσει σωστά.
Ορίσματα που δεν είναι αρχεία κειμένου ή δεν υπάρχουν στο σύστημα αγνοούνται από το πρόγραμμα.
Τα ορίσματα μπορούν να δηλώνονται είτε με relative είτε με absolute path.

**Τι κάνει η εφαρμογή**

Μόλις εκτελέσετε την εφαρμογή θα αρχίσει να εκτυπώνει τις συχνότητες των λέξεων των αρχείων κειμένου, μέχρι να τελειώσει και θα τερματίσει.
Όσο τρέχει η εφαρμογή δεν έχετε την δυνατότητα να την διακόψετε.
